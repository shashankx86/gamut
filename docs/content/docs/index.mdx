Gamut is a keyboard-driven application launcher for Linux. It runs as a lightweight background daemon
and surfaces a launcher overlay on demand — letting you find and open applications without reaching for the mouse.

## Key features

- **Instant toggle** — summon and dismiss the launcher with a single command or keyboard shortcut
- **Desktop entry search** — indexes `.desktop` files from standard XDG locations
- **Multi-monitor aware** — targets the display your cursor is currently on
- **System tray** — optional tray icon for quick access
- **Fully configurable** — theme, layout, and keyboard shortcuts are all customizable via a TOML config file

## How it works

Gamut runs a persistent daemon process in the background. When you want to launch an app, you send a
`toggle` command (via CLI, keyboard shortcut, or tray icon). The daemon opens the launcher window on
your active monitor. Type to search, navigate with the arrow keys, and press Enter to launch.

When the launcher is closed — by pressing Escape, launching an app, or losing focus — the daemon
keeps running so the next toggle is instant.
