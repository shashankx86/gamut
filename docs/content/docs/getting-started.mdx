## Requirements

- Linux (Wayland)
- A desktop environment that respects XDG standards

## Installation

### From source

You will need the [Rust toolchain](https://rustup.rs) installed.

```sh
git clone https://github.com/eopkg/gamut
cd gamut
cargo build --release
```

The compiled binary will be at `target/release/gamut`. Copy it somewhere on your `$PATH`:

```sh
sudo install -Dm755 target/release/gamut /usr/local/bin/gamut
```

## First run

Start the background daemon:

```sh
gamut --daemon
```

With the daemon running, toggle the launcher open:

```sh
gamut --toggle
```

The launcher window will appear on your active monitor. Type to search for an application, use
`ArrowDown` / `ArrowUp` to move through results, and press `Enter` to launch.

Press `Escape` to close the launcher. The daemon stays running so the next toggle is instant.

## Autostart

To have the daemon start automatically with your desktop session, enable the **Start at login**
setting with the config CLI, or set it in your config file:

```toml title="~/.config/gamut/config.toml"
[system]
start_at_login = true
```

```sh
gamut config set system.start_at_login true
```

On most desktops this works by writing a `.desktop` autostart entry to
`~/.config/autostart/gamut.desktop`.

## Configuration

Use the config CLI to inspect and adjust theme, layout, and shortcuts:

```sh
gamut config show
gamut config set appearance.theme dark
gamut config shortcut interactive
```

Changes are saved immediately to `~/.config/gamut/config.toml`.

## Stopping the daemon

```sh
gamut --quit
```

This sends a quit signal to the running daemon and exits cleanly.
