---
title: Configuration
description: Customize Gamut's appearance, layout, and keyboard shortcuts via the TOML config file or the config CLI.
---

Gamut stores all settings in a single TOML file:

```
~/.config/gamut/config.toml
```

The file is created with defaults on first run. You can edit it by hand or use the config CLI.

Below is a full annotated example with every available option and its default value.

```toml title="~/.config/gamut/config.toml"
[appearance]
# Color scheme to use: "system" | "light" | "dark"
# "system" follows the desktop's light/dark preference.
theme = "system"

# Corner radius preset: "small" | "medium" | "large" | "custom"
radius = "small"

# Radius in pixels when radius = "custom"
custom_radius = 10.0

# Light color scheme colors (hex, 6 or 8 digits)
[appearance.schemes.light]
background = "#F3F4F6"
text       = "#1F2328"
accent     = "#416EF5"

# Dark color scheme colors
[appearance.schemes.dark]
background = "#151516"
text       = "#EBEDF2"
accent     = "#5E8BFF"

[layout]
# Launcher size preset: "small" | "medium" | "large" | "extra_large"
size = "medium"

# Launcher placement: "center" | "raised_center" | "custom"
placement = "raised_center"

# Distance from the top of the screen in pixels when placement = "custom"
custom_top_margin = 120.0

[shortcuts]
# ====================================================================
# SHORTCUT CONFIGURATION
# To change shortcuts, use integer Virtual Keycodes.
#
# Format:
# action = [modifier_code, key_code]
# action = [key_code]
# action = [modifier_code, key_code, key_code, ...]
# ====================================================================

# Launch selected item
launch_selected = [13]

# Expand results
expand = [40]

# Move selection down
move_down = [40]

# Move selection up
move_up = [38]

# Close launcher
close_launcher = [27]

[system]
# Launch the daemon automatically when the desktop session starts
start_at_login = false
```

Common CLI workflows:

```sh
gamut config show
gamut config get appearance.theme
gamut config set layout.placement custom
gamut config set layout.custom_top_margin 96
gamut config shortcut set close-launcher [17,81]
gamut config shortcut interactive
```

---

## Appearance

### `theme`

Controls which color scheme is active.

| Value | Description |
|---|---|
| `"system"` | Follow the desktop light/dark mode (default) |
| `"light"` | Always use the light scheme |
| `"dark"` | Always use the dark scheme |

### `radius`

Corner radius of the launcher window.

| Value | Description |
|---|---|
| `"small"` | Subtle rounding (default) |
| `"medium"` | Moderate rounding |
| `"large"` | Heavy rounding |
| `"custom"` | Use the `custom_radius` value |

### `schemes.light` / `schemes.dark`

Each scheme defines three hex colors:

| Field | Description |
|---|---|
| `background` | Window background color |
| `text` | Primary text color |
| `accent` | Accent / highlight color |

Colors must be 6-digit (`#RRGGBB`) or 8-digit (`#RRGGBBAA`) hex strings.

---

## Layout

### `size`

Controls the overall size of the launcher window.

| Value | Description |
|---|---|
| `"small"` | Compact |
| `"medium"` | Default |
| `"large"` | Spacious |
| `"extra_large"` | Maximum |

### `placement`

Where the launcher appears on the screen.

| Value | Description |
|---|---|
| `"center"` | Exact vertical center of the screen |
| `"raised_center"` | Slightly above center (default) |
| `"custom"` | Use `custom_top_margin` to set the top offset |

### `custom_top_margin`

Distance in pixels from the top of the screen when `placement = "custom"`. Defaults to `120.0`.

## Shortcuts

Each shortcut is an array of integer Virtual Keycodes.

```toml
[shortcuts]
launch_selected = [13]
```

### Format

| Pattern | Description |
|---|---|
| `[key_code]` | Single-key shortcut |
| `[modifier_code, key_code]` | One modifier plus key |
| `[modifier_code, key_code, key_code, ...]` | One modifier plus multiple trigger keys |

### Common modifier codes

| Key | Code |
|---|---|
| `Shift` | `16` |
| `Ctrl` | `17` |
| `Alt` | `18` |
| `Super` / `Meta` | `91` |

### Common key codes

| Key | Code |
|---|---|
| `Enter` | `13` |
| `Escape` | `27` |
| `ArrowUp` | `38` |
| `ArrowDown` | `40` |

### Configurable actions

| Setting | Default | Description |
|---|---|---|
| `launch_selected` | `[13]` | Launch the focused application |
| `expand` | `[40]` | Expand results |
| `move_down` | `[40]` | Move selection down |
| `move_up` | `[38]` | Move selection up |
| `close_launcher` | `[27]` | Close the launcher |

---

## System

### `start_at_login`

When `true`, Gamut writes an autostart `.desktop` entry to `~/.config/autostart/gamut.desktop` so
the daemon is started automatically when your desktop session begins.

```toml
[system]
start_at_login = true
```
